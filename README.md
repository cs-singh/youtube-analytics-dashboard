# YouTube Analytics Dashboard

## Overview
The YouTube Analytics Dashboard provides insights into your YouTube channel's performance through various metrics and visualizations, enabling creators to make informed decisions and enhance their audience engagement.

## Features
- **Real-time statistics**: View live metrics of your channel’s performance.
- **Historic data analysis**: Analyze previous trends and performance over time.
- **Engagement rates**: Metrics showing how well your content is resonating with viewers.
- **Customizable dashboards**: Users can modify and save layouts tailored to their specific needs.

## Getting Started
1. **Installation**: Clone the repository and install the necessary dependencies.
   ```bash
   git clone https://github.com/cs-singh/youtube-analytics-dashboard.git
   cd youtube-analytics-dashboard
   npm install
   ```  

2. **Configuration**: 
   - Set up your YouTube API credentials in the `.env` file. Follow the YouTube API documentation for instructions on acquiring your API key.
   
3. **Run the Dashboard**: Start the application using:
   ```bash
   npm start
   ```

## Usage
- Navigate to the home page to view your dashboard.
- Use the navigation bar to switch between different metrics and visualization types.
- Filter data by date, video, and other criteria to gain specific insights.
